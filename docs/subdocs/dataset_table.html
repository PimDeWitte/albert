Centralized Dataset Registry

All datasets used in the gravity compression framework with full provenance tracking

Dataset Management

All datasets are managed through the central DatasetLoader class. Each dataset has:

  • A unique URI: dataloader://dataset_id
  • Source tracking in source.txt files
  • Automatic download and caching for remote datasets
  • Version control through timestamps

Usage Example

from physics_agent.dataset_loader import get_dataset_loader

# Load a dataset
loader = get_dataset_loader()
dataset = loader.load_dataset('planck_cmb_2018')

# Access the data
data = dataset['data']  # NumPy array or dict
metadata = dataset['metadata']  # Full dataset info
uri = dataset['uri']  # dataloader://planck_cmb_2018
