How to Use

New Way (Recommended)

from physics_agent.unified_quantum_solver import UnifiedQuantumSolver

# Create solver with PennyLane enabled (default)
solver = UnifiedQuantumSolver(theory, use_pennylane=True)

# Or without PennyLane
solver = UnifiedQuantumSolver(theory, use_pennylane=False)

# Compute trajectory
result = solver.compute_trajectory(start, end, method='wkb')
amplitude = result['amplitude']
trajectory = result['quantum_trajectory']

Old Way (Still Works)

# This still works for backward compatibility
from physics_agent.quantum_path_integrator import QuantumPathIntegrator

solver = QuantumPathIntegrator(theory)  # Redirects to UnifiedQuantumSolver
